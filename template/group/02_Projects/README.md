# Projects

Active, time-bound initiatives with clear completion criteria.

## Structure
Each project gets its own folder with:
- `README.md` - Objectives, timeline, status
- `Research/` - Source materials and literature notes
- `Drafts/` - Work in progress
- `Output/` - Final deliverables

## Lifecycle
1. Create project folder with README
2. Link to relevant Zettelkasten notes
3. Track progress in README
4. Archive when complete (move to 04_Archive/)
