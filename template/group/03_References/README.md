# References

Literature notes, source materials, and external content.

## Types
- **Literature notes** - Summaries of books, papers, articles
- **Web clippings** - Saved web content (via Firecrawl or manual)
- **Source materials** - PDFs, documents, data

## Naming
`Reference - [Title] - [Author/Source].md`

## Workflow
1. Capture source material here
2. Create literature notes summarizing key points
3. Extract atomic ideas into Zettelkasten notes
4. Link back to the source from your Zettelkasten notes
